# Rock Paper Scissors Game

A simple Rock Paper Scissors game implemented using HTML, CSS, and JavaScript. Test your luck against the computer and see who wins in this classic game!

## Getting Started

To play the game, open the `index.html` file in your web browser. Click on the "ROCK", "PAPER", or "SCISSORS" buttons to make your choice and see the result against the computer.

## Prerequisites

There are no specific prerequisites for this game. You only need a web browser to play the game.

## How to Play

1. Open the `index.html` file in your web browser.
2. Click on one of the buttons ("ROCK", "PAPER", or "SCISSORS") to make your choice.
3. The computer will randomly select one of the options as well.
4. The game will display the result - whether you win, lose, or it's a draw.

## Built With

- **HTML:** The structure of the game interface is created using HTML.
- **CSS:** The styling of the game elements is defined in the `style.css` file.
- **JavaScript:** The game logic is implemented using JavaScript in the `script.js` file.
